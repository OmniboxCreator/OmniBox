# Gerber Package

This package was regenerated with KiCad 10.0.4 from the corrected PCB. `D_PWR1`
uses the definitive `SM8S24A` orientation: pad 1 is the small cathode lead on
`VBAT_PROT`, and pad 2 is the large anode tab on `GND_V`.

This directory contains the local-finish board 1.0 beta manufacturing package:

- `gerber/`: Gerber and drill files, including the drill report.
- `j2534-gerbers-jlcpcb.zip`: zipped Gerber package.
- `j2534-local-finish-scripted.kicad_pcb`: fabrication PCB snapshot.

Review stack-up, drill files, solder mask and paste layers with the PCB vendor
before ordering.
