# Support OmniBox

OmniBox is public and free to use for non-commercial work. If the project helps
you, financial support is welcome and helps fund continued development.

## Donations and Sponsoring

One-time donations and recurring sponsorship help fund:

- prototype runs;
- components and replacement parts;
- test fixtures;
- bench ECUs and validation hardware;
- measurement equipment;
- documentation and release work.

Suggested channels:

- GitHub Sponsors
- Ko-fi
- Liberapay
- PayPal
- direct sponsorship by agreement

Add the active links here once they are available.

## Commercial Licensing

OmniBox uses a dual-license model. The public repository is free for
non-commercial use. Commercial use requires written permission.

Contact the project maintainer if you want to:

- manufacture or distribute hardware derived from OmniBox as part of a business;
- use OmniBox in a paid service;
- integrate OmniBox into a commercial product;
- distribute modified versions commercially;
- include OmniBox in a paid product, training package or business offering.

Commercial licensing can be discussed case by case.

## Contributions

Contributions are welcome in these areas:

- board review;
- protection and isolation review;
- firmware development;
- J2534 implementation and tests;
- ELM327 compatibility;
- host tools and DLL work;
- vehicle and bench ECU validation;
- documentation;
- enclosure and mechanical improvements.

If you want to contribute but are unsure where to start, open an issue or contact
the maintainer with your skills and available hardware.
